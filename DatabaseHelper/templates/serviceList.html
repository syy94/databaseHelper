APIS:

For Post, use something like Postman to test the service
Check http status for success of the service call:

POST

  • list

    • JsonObject containing:

      • length: int
      • offset: int
      • school: string | optional
      • order: ["id", "name", "poly", "score", "course_type"]
        (optional)

    • Returns: JSONArray
  • course

    • JsonObject containing:

      • id: string

    • Returns: course:JSONObject (if not found, course:JSONObject will not exist)
  • search

    • JsonObject containing:

      • query: string (string of keywords, if multiple, separate by a space ' '. Eg. "sci app")
      • offset: int (for loading more search results)

    • Returns: JsonArray
